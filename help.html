üêç Python Control

Write Python code in the editor to control R2-D2 using the SITH API module.

Getting Started:

  • Write Python code in the code editor
  • Use the sith_api module to control R2-D2
  • Click 'Run' to execute your code
  • View output in the console below

Available Functions:

  • sith.send_command(cmd) - Send raw Shadow command
  • sith.open_panel(num) - Open specific panel (1-16)
  • sith.close_panel(num) - Close specific panel (1-16)
  • sith.open_all_panels() - Open all panels
  • sith.close_all_panels() - Close all panels
  • sith.run_sequence(name) - Run sequence by name
  • sith.get_status() - Get R2-D2 status
  • sith.get_panels() - Get panel states
  • sith.reset() - Reset R2-D2
  • sith.set_servo(num, pos) - Set servo 1-8 to position 0-180
  • sith.set_dome_motor(dir, speed) - Set dome motor (CW/CCW/STOP)
  • sith.dome_rotate_cw(speed) - Rotate dome clockwise
  • sith.dome_rotate_ccw(speed) - Rotate dome counter-clockwise
  • sith.dome_stop() - Stop dome rotation

üìö Shadow Protocol Commands

Complete reference for Shadow protocol commands used by R2-D2.

Panel Commands

:OP01-16 Open specific panel
:CL01-16 Close specific panel
:OP00 Open all panels
:CL00 Close all panels

Sequence Commands

:SEwave Wave sequence
:SEscream Scream sequence
:SEdance Dance sequence
:SEmarch March sequence

Holo Projector

*H001-008 HP effects

Display Commands

@0T1-8 Display patterns

Sound Commands

$A-H Sound effects
$S Scream sound

Setup Commands

#RST Reset
#CAL Calibration
#TST Test
#VER Version
#STA Status
